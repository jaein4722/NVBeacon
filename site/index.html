Why NVBeacon

It answers the questions GPU users keep asking.

Is the box busy?

Menu bar summary, configurable busy detection, and a compact popover make server load visible at a glance.

Which job is mine?

Per-GPU process detail shows user, PID, GPU memory, and command preview without keeping a shell open.

Can I reuse this GPU?

Star a GPU to watch for long idle periods and get notified when it becomes a candidate for the next run.

Did my job finish?

Track specific processes and get a native macOS notification when they really exit.

Quick start

Install, point at a server, and monitor from the menu bar.

  1. Install NVBeacon from GitHub Releases or Homebrew.
  2. Open Settings from the menu bar icon.
  3. Set an SSH target directly or import from ~/.ssh/config.
  4. Choose key-based or password-based authentication.
  5. Allow notifications if you want process exit or GPU idle alerts.

Requirements

  • macOS 14 or later
  • SSH access to the target machine
  • nvidia-smi available on the remote host
